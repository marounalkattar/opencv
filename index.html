Introduction

Color detection

In this mini-project, I will detect colors by creating an HSV window in which I put the max and minimum Hue, Saturation, and result.

Tracking object

In this section, I have made an object tracking to track the action camera so it will tell me how much fps there is and if I’m really tracking or have I lost it.

Faces detection

I used a method proposed by VIOLA & JONES so instead of training the model I have used a present file for faces which is provided by OpenCV.
